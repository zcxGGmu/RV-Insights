# UI Screenshots

本目录保存 RV-Insights 客户端 UI 优化阶段的审计与验收截图。

## Pipeline UI-3

- `pipeline-ui3-light-desktop.png`：Pipeline 浅色主题，展示工作台主布局、StageRail、Records 和 Composer。
- `pipeline-ui3-dark-desktop.png`：Pipeline 深色主题，展示运行状态、Live output 和右侧操作区。
- `pipeline-ui3-slate-light-desktop.png`：Pipeline 特殊主题，验证状态层级和主题 token 适配。
